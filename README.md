# GoWorkout

GoWorkout is an online fitness platform with customizable workout plans designed to help you achieve your health and training goals â€” anytime, anywhere.

## Features

GoWorkout provides users with:
- **User Authentication**: Sign up and login functionality with secure password management
- **Training Programs**: Three levels of workout programs (Beginner, Intermediate, Advanced) with detailed exercise routines
- **Calorie Calculator**: Calculate daily calorie needs based on weight, height, age, gender and fitness goals

## Prerequisites

To run this application, you need:
- Node.js (v14 or higher)
- npm (Node Package Manager)

## Installation & Setup

1. **Extract the project** to your desired directory

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the server**:
   ```bash
   node server.js
   ```

4. **Open in your browser**:
   - Navigate to `http://localhost:3000`

## Getting Started

1. Sign up for an account or log in
2. Use the Calorie Calculator to determine your daily calorie needs
3. Explore training programs suited to your fitness level
